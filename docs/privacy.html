⚡ TokenSaver — Privacy Policy

Effective date: February 2026 · github.com/shuaills/tokensaver

TL;DR: TokenSaver does not collect, transmit, or store any personal data. Everything happens locally in your browser.

What TokenSaver does

TokenSaver is a Chrome extension that intercepts paste events on LLM chat sites (Claude, ChatGPT, Gemini, etc.) and removes redundant whitespace and formatting noise from pasted text to reduce token usage. All text processing happens entirely within your browser using JavaScript.

Data we collect

None. TokenSaver does not collect any personal information, browsing history, or content of any kind.

Local storage only

The extension stores two things locally on your device using the Chrome storage API:

  • Preferences — your chosen cleaning intensity (soft or aggressive) and enabled/disabled state, synced across your Chrome profile via chrome.storage.sync.
  • Session stats — a running count of characters saved, shown in the extension popup, stored via chrome.storage.local.

This data never leaves your device and is never transmitted to any server.

No network requests

TokenSaver makes zero network requests. It has no backend, no analytics, no crash reporting, and no telemetry of any kind. You can verify this by inspecting the source code on GitHub.

Permissions explained

  • storage — to save your preferences and local usage stats.
  • Host permissions (claude.ai, chatgpt.com, etc.) — to inject the content script that intercepts paste events on supported sites. The extension only activates on these specific domains.

Third parties

TokenSaver does not share data with any third party because it does not collect any data in the first place.
